# KB-004 内部依赖错误

## 故障现象
日志中出现 'internal dependency error' 或 'Failed to call'

## 涉及服务
order-service → product-service/payment-service

## 根因分析
下游服务出现故障或响应超时

## 影响范围
上游服务无法完成请求，导致级联失败

## 处理步骤
1. 从日志中提取 Trace ID
2. 通过链路追踪定位具体故障服务
3. 检查故障服务的日志和指标
4. 确认故障服务恢复后重新尝试

## 预防措施
1. 实现服务熔断机制
2. 配置合理的超时时间
3. 实施降级策略
