# KB-003 支付网关超时故障

## 故障现象
日志中出现 'Gateway timeout' 或 'Payment gateway 503 Service Unavailable'

## 涉及服务
payment-service → external-payment-gateway

## 根因分析
外部支付提供商服务宕机、网络延迟过高或触发了限流机制

## 影响范围
支付流程中断，用户无法完成付款

## 处理步骤
1. 检查支付提供商官方状态页面确认是否宕机
2. 使用指数退避策略重试交易
3. 考虑切换备用支付网关
4. 在 payment-service 中增加超时和熔断逻辑

## 预防措施
1. 配置合理的超时时间
2. 实现熔断器模式
3. 准备备用支付渠道
