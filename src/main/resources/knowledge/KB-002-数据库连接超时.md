# KB-002 数据库连接超时故障

## 故障现象
日志中出现 'Connection to InventoryDB timed out' 或 'Connection timeout'

## 涉及服务
product-service → inventory-db

## 根因分析
高负载导致数据库响应缓慢，或网络拥塞导致连接超时

## 影响范围
product-service 无法获取库存数据，导致结账流程中断

## 处理步骤
1. 检查数据库 CPU 使用率，若超过 90% 则需要扩容
2. 检查是否存在慢查询或锁竞争
3. 检查网络延迟和连接池配置
4. 优化查询语句或添加索引

## 预防措施
1. 设置合理的连接超时时间
2. 实现连接池监控
3. 定期优化慢查询
